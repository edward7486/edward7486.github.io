Design System

Setting up the foundations of a design system in conjunction with the front-end development team.

Overview

  • Company Skilljar
  • Timeline 6 months
  • Methods Visual Design, Design Systems
  • Team Edward Nguyen, Ian Johnson, Chip Cressman, Aizat Murtaza, Gregory Buehler

Methodology

As part of a product-wide visual refresh, I created a design system within Sketch (and eventually Figma) that laid out the core styles and components to be utilized for designs going forward. This helps designers pull from a UI library to ensure consistency.

To get started, I first created a catalog within MindManager of each page, and within each page the layouts and components used. This yielded an inventory of components and their counts, allowing me to begin defining styles as needed.

I worked closely with lead front-end engineers to solidify core styling, grids, breakpoints, and components. It was my first time creating a design system and a great learning experience.
